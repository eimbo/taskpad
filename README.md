# TaskPad

TaskPad is an online text editor created by developers for developers. 

The application data is stored in the browser and nothing is sent across the internet.

This project was generated with Angular CLI version 15.1.3. [Angular](https://angular.io).

## Contributors

Download the project and install dependencies with `npm install`.

Run `ng serve` for a dev server. Navigate to `http://localhost:4200/`. 

Run `ng build` to build the source files.
