# TaskPad

Visit the project at www.taskpad.dev.

TaskPad is an onine text editor created for developers. The application data is stored in the browsers local storage and nothing is sent over the internet. This project was started to learn Angular and contributions are welcome. 



This project was generated with Angular CLI version 15.1.3. [Angular](https://angular.io).

## Contributors

Consider contributing if you would like to help out and learn Angular on an open source project. Download the project and install dependencies with `npm install`, then run `ng serve` for a dev server. The static source files are deployed to an Amazon S3 bucket where the content is served via a CloudFront distrubution.
